# Pokedex React Web App

## Description

This is a React-based web application that utilizes the [nestjs-pokedex-api](https://github.com/dalemncy/nestjs-pokedex-api) API to fetch and display Pokemon-related data.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```
